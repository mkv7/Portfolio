A minimal, no-build portfolio site for **Moaeez Muhammad**. Uses React 18 from a CDN so you can host on **GitHub Pages** or any static host without Node tooling.

## Features
- Dark, minimal layout
- Sections: About, Education, Projects, Contact
- Single HTML + JS + CSS (no bundler)

## How to run
Just open `index.html` in a browser, or serve the folder with any static server.

For example (Python 3):
```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Customize
- Edit text in `main.js` (About, Education, Projects, Contact)
- Edit colors and spacing in `styles.css`
- Update links to your GitHub repos when those are live
